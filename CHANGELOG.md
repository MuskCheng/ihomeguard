# 更新日志

本文档记录 iHomeGuard 的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.2.15] - 2026-03-10

### 修复
- **日报流量统计修正**：网络日报流量数据由累计值改为真正的日增量计算
  - 计算方式：当日最后记录 - 前一日最后记录 = 当日增量
  - 修复日期参数传递问题，确保日报发送正确日期的数据
- **敏感字段脱敏**：设置页面中的 push_key、bot_token、secret 等敏感字段正确显示为脱敏格式
- **页面刷新保持登录**：修复刷新页面后需重新登录的问题，从 localStorage 恢复认证状态
- **告警日期过滤**：日报中的告警改为只显示当日的告警

### 优化
- **JWT Token 有效期**：从 24 小时改为 2 小时，提升安全性
- **日报内容增强**：
  - 新增流量环比对比（与前一日对比，显示涨跌幅）
  - 新增在线时长 TOP5 排行
  - 优化日报格式和展示效果

## [1.2.12] - 2026-03-10

### 修复
- **今日流量统计修正**：设备列表中的"今日流量"原为设备累计流量，现改为真正的今日增量计算
  - 计算方式：今日最后记录 - 昨日最后记录（或今日首条记录）
  - 新增 `storage.get_device_today_traffic()` 和 `storage.get_all_today_traffic()` 函数

### 优化
- **设备名称显示优化**：所有通知推送中设备名称优先级改为 `备注名 > 主机名 > MAC前8位`
  - 新增 `alerter._get_device_display_name()` 辅助方法统一处理
  - 新设备接入通知新增"设备名称"字段作为主要展示项
- **配置项调整**：
  - 默认采集间隔从 5 秒改为 10 秒
  - 会话超时配置从"监控策略"移至"连接配置"区块（与爱快系统账号配置同组）
- **设置页面布局优化**：会话超时字段移至连接配置，监控策略区块更简洁

### 文档
- README 更新爱快路由器配置说明，新增会话超时配置项说明

## [1.2.3] - 2026-03-09

### 新增
- **安全认证**：可选 Token 认证保护，支持环境变量 `AUTH_TOKEN` 或配置文件启用
- **敏感字段脱敏**：push_key、webhook 等敏感信息在界面显示为脱敏格式
- **备份恢复**：支持配置文件、设备设置、告警记录的导出导入
- **版本检查**：设置页面新增"关于"区块，显示版本信息和更新日志
- **统一日志系统**：新增 `logger.py` 模块，支持 `LOG_LEVEL` 环境变量控制日志级别

### 修复
- 启动通知防重复机制：60秒内不重复发送启动通知

### 优化
- 所有 `print()` 语句替换为标准 `logging` 模块
- 异常处理优化，更精确的错误类型捕获
- 认证状态持久化（localStorage 保存 token）

## [1.2.1] - 2026-03-07

### 修复
- 新设备接入告警逻辑：只在设备首次接入时触发，非信任设备重复上线不再误报
- 系统启动通知频繁发送问题：新增开关控制

### 新增
- `alert_startup` 配置项：控制系统启动通知开关（默认启用）
- `alert_offline` 配置项：显式控制信任设备离线通知开关
