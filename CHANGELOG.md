# 更新日志

本文档记录 iHomeGuard 的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.2.1] - 2026-03-07

### 修复
- 新设备接入告警逻辑：只在设备首次接入时触发，非信任设备重复上线不再误报
- 系统启动通知频繁发送问题：新增开关控制

### 新增
- `alert_startup` 配置项：控制系统启动通知开关（默认启用）
- `alert_offline` 配置项：显式控制信任设备离线通知开关

## [1.2.0] - 2026-03-07

### 修复
- 移动端页面点击无反应问题（移除 chartjs-plugin-zoom 插件、修复遮罩层拦截事件）
- 连接数监控图表无数据问题（修复图表渲染空指针异常）

### 新增
- 流量趋势调整为最近14天数据
- 连接数数据改为每分钟最后一条记录

### 优化
- 数据保留期从7天改为24小时
- 统一所有页面标题样式（图标+文字左对齐）
- 刷新按钮统一居右布局
- 移除实时流量监控的LIVE徽章和刷新按钮
- 移除设备在线统计图表（简化界面）

## [1.1.0] - 2026-03-06

### 修复
- 统计页面流量计算逻辑（累计值改为增量计算）
- 配置保存后数据不刷新问题
- 月份天数硬编码问题（改用 calendar.monthrange）
- 数据库连接泄漏风险（vacuum_database 添加 finally）
- 裸 except 异常处理（改为 except Exception as e）

### 优化
- 统计页面实时刷新优化（5秒流量历史、30秒统计汇总）
- 连接测试成功后自动刷新设备数据
- 告警页面添加 10 秒定时刷新

### 文档
- 更新 README 监控参数说明与代码一致
- 添加版本标签徽章

## [1.0.0] - 2026-03-05

### 新增
- 设备实时监控功能
- 流量统计与趋势图表
- 智能告警系统
- 多渠道消息推送（PushMe、企业微信、钉钉）
- 日间/夜间主题切换
- 炫酷现代化 UI
- 数据加密存储
- 月度流量预测
