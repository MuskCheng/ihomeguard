# 更新日志

本文档记录 iHomeGuard 的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.1.0] - 2026-03-06

### 修复
- 统计页面流量计算逻辑（累计值改为增量计算）
- 配置保存后数据不刷新问题
- 月份天数硬编码问题（改用 calendar.monthrange）
- 数据库连接泄漏风险（vacuum_database 添加 finally）
- 裸 except 异常处理（改为 except Exception as e）

### 优化
- 统计页面实时刷新优化（5秒流量历史、30秒统计汇总）
- 连接测试成功后自动刷新设备数据
- 告警页面添加 10 秒定时刷新

### 文档
- 更新 README 监控参数说明与代码一致
- 添加版本标签徽章

## [1.0.0] - 2026-03-05

### 新增
- 设备实时监控功能
- 流量统计与趋势图表
- 智能告警系统
- 多渠道消息推送（PushMe、企业微信、钉钉）
- 日间/夜间主题切换
- 炫酷现代化 UI
- 数据加密存储
- 月度流量预测
